# AI in Primary Care: A Bibliometric & Methodological Analysis (1995-2025)

This repository contains the data, code, and supplementary materials for the study: 
**"Mapping the Intellectual Landscape of Artificial Intelligence in Primary Care: Global Trends, Intellectual Structures, and Methodological Maturity"**

## 📂 Contents
- `/Data`: Merged bibliometric metadata (WoS, Scopus, PubMed).
- `/Scripts`: R code for Bibliometrix analysis and Cohen's Kappa bootstrapping.
- `/Supplements`: LLM extraction prompts (Gemini 3.1 Pro) and TRIPOD-AI scoring rubric.

## 🛠️ Tools Used
- **R v4.5.3** (Bibliometrix v4.1, irr package)
- **LLM:** Gemini 3.1 Pro (Extraction Rater 1)
- **Human Expert:** Assoc. Prof. [Adınız] (Validation Rater 2)
